用hexo搭建博客网站过程说明。

私以为，博客是一种有效的记笔记方式，方便回顾，同时，博客也是交流知识的平台，与他人互相学习，共同进步。机缘巧合，了解到可以利用hexo和github搭建博客网站，便打算试一试，先用着看看效果如何吧。

预备知识

hexo是一个快速、简洁且高效的博客框架，使用Markdown（或其它渲染引擎）来解析文章，生成静态网页，此后，将网页推送到github平台。因此，主要用到以下工具，较为简单：

  1. github用法
  2. markdown语法

配置hexo

配置hexo是通过修改blog/_config_yml文件完成的。

为了将hexo和github关联起来，将deploy项修改为：

deploy:
type: git
repo: github上刚创建的仓库的完整路径，最后加上.git
branch: master

此外，根据官方文档，配置网站标题、描述、主题等等。
